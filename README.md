# league-hack

External hack for league of legends (py version rebuild)

Features
-----
* Aim-Lock
  * FOV
  * ~~Smooth & Delay~~
  * Movement Predict
* ~~ESP~~
  * ~~AARange~~
  * ~~Snaplines~~
  * ~~2Dbox~~
  * ~~Predict Lines~~
  * ~~Stream Mode (game capture)~~
* Orbwalker
  * WindUpDelay

Game Settings
-----
* FPS : +144
* Colorblind Mode : on
* Show Life Bar : on
* Cam Mode : fixed
* Movement With Attack Set 2 : 'L'
* Bind Auto Attack Move to left-click : off
* Window Mode (to ESP)
